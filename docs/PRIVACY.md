## 1. Data we collect

| Category | Data | Purpose | Legal basis |
|----------|------|---------|-------------|
| Account | Handle, email, display name, password hash | Account management | Contract (Art. 6(1)(b)) |
| Content | Photos/videos, captions, timestamps | Core service | Contract (Art. 6(1)(b)) |
| Attestation | Media hash, capture metadata, attestation score | Platform integrity | Legitimate interest (Art. 6(1)(f)) |
| Session | Refresh token hash, login timestamps | Security | Legitimate interest (Art. 6(1)(f)) |
| Technical | IP address (in logs, TTL 7 days), User-Agent | Security, abuse prevention | Legitimate interest (Art. 6(1)(f)) |

We do **NOT** collect:
- Location data (GPS or otherwise)
- Device identifiers beyond session scope
- Third-party tracking pixels or cross-site data

## Deutsche Zusammenfassung (German Summary)

**Datenschutzerklärung — Kurzfassung**

VERIDA ist eine soziale Web-App für spontane Tagesmomente. Wir erheben nur die Daten, die für den Betrieb des Dienstes erforderlich sind.

**Was wir speichern:**
- Nutzername, E-Mail-Adresse, Anzeigename (verschlüsseltes Passwort)
- Hochgeladene Fotos/Videos, Bildunterschriften
- Technische Metadaten zur Echtheitsprüfung (kein GPS, keine Geräte-IDs)
- IP-Adressen in Logs (7 Tage Aufbewahrungsfrist)

**Ihre Rechte nach DSGVO:**
Sie haben das Recht auf Auskunft (Art. 15), Berichtigung (Art. 16), Löschung (Art. 17), Datenübertragbarkeit (Art. 20) und Widerspruch (Art. 21).

Alle Rechte können über die Einstellungen oder per E-Mail an **privacy@verida.example** ausgeübt werden. Anfragen werden innerhalb von 30 Tagen bearbeitet.

**Weitergabe:** Wir verkaufen keine Daten. Keine Werbecookies. Keine Drittanbieter-Tracker.

**Datenlöschung:** Bei Kontolöschung werden alle personenbezogenen Daten innerhalb von 30 Tagen gelöscht. Attestierungsprotokolle werden 2 Jahre für Prüfzwecke aufbewahrt.
